# Fonts Directory

Please place your ChickenSans font files here:

- `ChickenSans-Regular.ttf` (or .woff/.woff2)
- `ChickenSans-Bold.ttf` (or .woff/.woff2)

The fonts are already configured in `src/index.css` and will be automatically loaded when you place the files here.

Supported formats:
- `.ttf` (TrueType)
- `.woff` (Web Open Font Format)
- `.woff2` (Web Open Font Format 2)
